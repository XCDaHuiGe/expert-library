## 评估体系（v2.0）

| 等级 | 名称 | 内容 | 检查方式 |
|:---|:---|:---|:---|
| L1 | 信息完整度 | 8项指标全部达标，量化评分（满分100） | 自动检查 |
| L2 | 仿真发言测试 | 给定话题，评估发言是否"像本人"（4维度x25分） | 模拟测试 |
| L3 | 跨专家去同质化 | 同类型专家间进行正交检验 | 对比分析 |

## 防幻觉机制（v2.0新增）

| 规则 | 说明 |
|:---|:---|
| 事实必须有来源 | 每个事实必须有至少1个搜索来源支撑 |
| 无法验证则标注 | 无法验证的信息标注 `[待核实]` |
| MBTI需谨慎 | 历史人物MBTI标注 `[推测]` 并说明依据 |
| 语录需出处 | 无出处的语录标注 `[出处待查]` |
| 搜索来源URL化 | 所有来源必须包含可点击的URL链接 |

## 使用方式

1. **训练新专家**：激活 `expert-training` skill，按模板逐位训练
2. **续训**：检查 `training-progress.md` 找到断点，给 AI 发送 `续训 专家名`
3. **在蒸馏中使用**：将训练好的专家档案提供给 `book-distillation-v4` 的圆桌Agent
