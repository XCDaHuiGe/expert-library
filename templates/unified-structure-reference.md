# 统一结构参考模板

> 这是所有专家档案应遵循的统一结构。以 steve-jobs.md 为标杆。

## 标准章节顺序

```
# 专家姓名（英文名）专家档案

## 基本信息          ← 表格形式（属性|内容）
## 核心立场          ← 一句话加粗
## 经典著作/作品     ← 表格（著作|年份|核心贡献|链接）
## 核心理论/观点     ← 每个理论：关键概念+深层逻辑/核心阐述+经典案例+来源
## 历史语境          ← 时代背景表格 + 历史地位列表
## 跨领域贡献        ← 主要领域影响表 + 跨界影响表
## 现代解读          ← 当代意义列表 + 争议与重新诠释表
## 经典语录          ← 表格（序号|语录|出处|验证状态）
## 发言风格          ← 语气特点 + 常用句式表 + 论证方式 + 标志性表达
## 模拟发言          ← 2-3个场景，对话体（提问者-专家）
## 批评与争议        ← 每条批评含：批评者+批评内容+"X式回应"（第一人称角色扮演）
## 适用书籍类型      ← ✅/⚠️ 标记
## 在圆桌讨论中的角色 ← 擅长环节+典型质疑方向+与其他专家的关系
## 搜索来源          ← Markdown链接格式
## 元数据            ← 创建日期|数据来源|专家类型|训练方法
## 训练状态          ← 训练时间|训练者|质量评分|L1自检表|审核状态
## 质量自检（L1评估）← 评估维度表（信息完整度/来源可靠性/理论深度/批评平衡性/跨领域覆盖/语录准确性/模拟发言质量）
```
